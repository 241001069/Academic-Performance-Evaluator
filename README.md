# Academic Performance Evaluator

## Project Overview
Academic Performance Evaluator is a project developed using Azure DevOps and GitHub to manage academic activities such as GPA calculation, attendance tracking, notifications, and user management.

- Secure Login Authentication
- User & Role Management
- GPA Calculation
- Attendance Tracking
- Notification Alerts
- Sprint Planning
- Test Plans and Test Cases
- Load Testing
- Azure Pipelines Integration
